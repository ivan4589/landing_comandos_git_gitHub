Guía rápida de Git y GitHub

Flujo básico para subir tu proyecto a GitHub

Una guía visual, limpia y moderna para inicializar tu proyecto, hacer commits y publicarlo en GitHub.

01

Inicializar proyecto

Crea la carpeta del proyecto, entra en ella e inicializa Git.

Terminal 
mkdir proyecto-Git
cd proyecto-Git
git init
02

Primer commit local

Agrega los archivos al área de preparación y registra tu primer cambio.

Terminal 
git add .
git commit -m "Descripción de lo que hicimos"
03

Crear repositorio en GitHub

Ingresa a GitHub desde tu navegador y crea un repositorio remoto para vincular tu proyecto local.

04

Vincular repositorio remoto

Conecta tu proyecto local con el repositorio que creaste en GitHub.

Terminal 
git remote add origin URL_DEL_REPOSITORIO_REMOTO
05

Enviar el proyecto a GitHub

Haz el primer push para subir tu proyecto a la nube.

Terminal 
git push -u origin master
06

Actualizar con nuevos cambios

Cada vez que modifiques tu proyecto, repite este flujo.

Terminal 
git add .
git commit -m "nuevos cambios"
git push origin master
